# CasualPass 🎮
CasualPass is a premium, ad-free, and AI-enhanced casual gaming platform built for a seamless web experience.

## ✨ Key Features
- **Ad-Free Experience:** Zero interruptions for pure gameplay.
- **Modern UI:** Sleek glassmorphism design with fluid animations.
- **Personalization:** visual themes (example: Liquid Glass, Paper, Dark Neon).
- **Smart Settings:** Global control over sound effects and visual performance.

## 🕹️ Available Games
- **Chess:** Featuring CasualFish Engine with multiple difficulty levels and smooth drag-and-drop mechanics.
- **Snake:** Classic arcade action with customizable grid sizes, speeds, and high-score tracking.
- **Tic Tac Toe (XOX):** Advanced Minimax-based CasualX0X Engine (Easy to Impossible), local multiplayer, and a competitive leaderboard.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Advanced Grid & Custom Variables), Vanilla JavaScript.
- **Logic:** `chess.js` for move validation.
- **Persistence:** `localStorage` for saving themes, high scores, and user preferences.
